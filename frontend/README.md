# Admin Dashboard (React)

## Overview

A simple admin dashboard built with React to manage users.
Includes user listing, editing, and data persistence.

## Features (Current)

- Dashboard with summary cards
- Users page with edit functionality
- Reusable Edit User modal
- Shared edit logic via custom hook
- Global state management using Redux
- Data persistence via backend API

## Tech Stack

- React
- Redux Toolkit
- Custom Hooks
- Tailwind CSS
- Vite

## Project Structure (High-level)

- redux/ → global state & persistence
- hooks/ → reusable feature logic
- components/ → reusable UI components
- pages/ → route-level pages
- layout/ → layout components
- Modals/ → modal components
- data/ → static data files

## Getting Started

1. Clone the repo
2. npm install
3. npm run dev
